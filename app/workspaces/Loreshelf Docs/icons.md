# Icon Codes

The **codes** entered to the notecards are **automatically** **transformed** into icons.

| Code   | Icon |
| ------ | ---- |
| \[ \]  | [ ]  |
| \[x\]  | [x]  |
| \[-\]  | [-]  |
| \[/\]  | [/]  |
| \[?\]  | [?]  |
| \[.\]  | [.]  |
| \[+\]  | [+]  |
| \[!\]  | [!]  |
| \[i\]  | [i]  |
| \[\*\] | [*]  |

Press the _Backspace_ key if you want to return the transformation back to the text state.
